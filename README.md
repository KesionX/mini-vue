# ue3

ue3 构建自己的 vue3 进行学习，代码基于《vue.js 设计与实现》编写。这是一个小型的vue3响应式框架，实现vue3的核心原理与设计。

## feature

目前我正在开发：

- 帮助你快速了解vue的设计和实现原理。
- 带有完整实现代码与类型提示。
- 完整的测试脚本。
- 我目前正在编写文档，希望有更详细的实现文档。
